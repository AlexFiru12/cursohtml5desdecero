# Ejercicios

## 1) Instalar Sublime Text
Instalar [Sublime Text](https://www.sublimetext.com/) y en ```preferences->settings-user```comprobar que están estas tres líneas (sino añadirlas):
```
{
	"indent": 2,
	"tab_size": 2,
	"translate_tabs_to_spaces": true
}
```
